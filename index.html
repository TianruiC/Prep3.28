  • Showing the consistent and periodic fluctuations about the data

Index Origin

  • Better represent the relative change
  • The y-axis is the percentage value. The zero percentage usually is at the middle of the y-axis
  • Percentage change = ((Each score – Index origin) / Indexorigin) × 100

Logarithmic Scale

  • Easier to visualize the data with expontential rate of change
  • Using log transformation to the value in y-axis to better represent the data. The majority case is to use log10 scale.

Horizon Chart

  • Good technique for displaying dozens of line charts simultaneously
  • Steps to make space-efficient horizon chart
    1. Subdivide its vertical scale evenly, and colors code positive and negative values
    2. Mirror the color bands corresponding to negative values
    3. then collapses the color bands
